## Style Guidelines

### Icon Design Principles
1. **Consistent stroke weight**: 2-3px for all line icons
2. **Rounded corners**: 2-4px radius for friendly feel
3. **Size**: Design at 24x24px base, scalable to 48x48px
4. **Padding**: Maintain 2px internal padding
5. **Accessibility**: Ensure 4.5:1 contrast ratio minimum

### Background Image Guidelines
1. **Overlay**: Use 40-60% dark overlay on photos for text readability
2. **Focus area**: Keep main subject in rule-of-thirds composition
3. **Color temperature**: Warm (3000K-5000K) for comfort, cool (5500K-7000K) for clarity
4. **Blur/Bokeh**: Use when text overlay needed, maintain subject focus
5. **Resolution**: Minimum 1920x1080px for web, 4K for print

### Typography Pairing Suggestions
- **Headers**: Open Sans Bold, Montserrat SemiBold, or Roboto Medium
- **Body**: Open Sans Regular, Lato Regular, or Source Sans Pro
- **Accent/Quotes**: Merriweather, Playfair Display (for warmth)
- **Accessibility**: Minimum 16px body text, 1.5 line height

### Emotional Tone
- **Empowering, not pitying**: Show strength and resilience
- **Welcoming, not clinical**: Warm colors and organic shapes
- **Modern, not corporate**: Clean but human-centered
- **Hopeful, not naive**: Realistic optimism, grounded in lived experience

## Implementation Notes

### For Call-to-Action Backgrounds
- Use overlay text in white or cream with 60% dark gradient
- Position CTAs in lower-third or center-left for balance
- Ensure mobile responsiveness (test at 375px width)
- Consider loading times - optimize to <200KB per image

### For Icons
- Export as SVG for web scalability
- Provide PNG at 1x, 2x, 3x for various displays
- Create icon font option for performance
- Include alt text descriptions for accessibility
- Consider animated versions for key actions (subtle, 0.3s transitions)

### Brand Consistency
- Create a digital style guide with all approved variations
- Limit icon color palette to 4-5 core colors + neutrals
- Test all visuals with community members and clients
- Regular review (quarterly) for cultural sensitivity and relevance
